Overview

모니터링 업무는 운영자나 개발자가 근무 시간에 이용하기 때문에 주로 데스크톱에서 웹 서비스를 통해 접속합니다. 하지만 근무 외의 시간에도 기업 서비스에 장애가 생길 수 있기 때문에 접근성이 높은 모바일로 중요한 데이터들을 언제 어디서든 쉽게 살펴볼 수 있습니다.

Project. 등록한 프로젝트의 중요 정보를 한눈에.

기업의 서비스 규모가 클수록 프로젝트의 갯수가 많이 늘어나기 때문에 중요한 데이터를 선별하여 많은 프로젝트들을 쉽게 볼 수 있도록 구성하였습니다. 상단에는 카테고리를 필터링해서 정렬해주는 기능을 담아 정보를 빠르게 찾을 수 있도록 하였고, 프로젝트에는 어떤 프로젝트인지 알려주는 제품태그와 이벤트의 유무, 중요 지표들을 담았습니다.

Project Detail. 프로젝트의 현재 상태와 과거의 기록까지 보여주는 상세페이지

대시보드를 통해 프로젝트의 상태를 시간 범위에 따른 그래프로 알 수 있으며, 서버 목록에서는 프로젝트에 등록된 서버별 현재 상태를 알 수 있습니다. 이벤트 페이지에서는 어떤 서버에서 이벤트가 남아있는지 확인하여 문제점을 빠르게 확인할 수 있습니다.

Time Setting. 빠르게 원하는 범위로 시간 설정.

시간 설정 부분은 실시간 그래프를 우선적으로 보여주어 현재 프로젝트 상태를 알려줍니다. 그리고 사용자들이 많이 사용하는 시간 범위를 선정하여 선택할 수 있도록 설정하였고, 원하는 선택지가 없다면 커스터마이징을 통해 설정할 수 있습니다.
